# Project Status

This repository is a portfolio v0.

It is intended to show a professional BI and performance reporting workflow using processed and anonymized data. It is not presented as a complete production dashboard or enterprise data platform.

## Completed

- Processed and anonymized reporting dataset
- Python ETL pipeline
- Asset-level performance summary
- Power BI model notes and DAX measure library
- DAX measures for the selected report pages
- Budget-vs-actual variance logic
- Wind versus residual attribution logic
- Data quality and exclusion flags
- Board review report page
- Asset-level variance detail page
- Reconciliation appendix screenshot
- Data quality appendix screenshot

## In Progress

- Final visual polish
- Additional report navigation
- Final DAX naming and formatting cleanup before any later PBIX release
- Screenshot-based walkthrough
- Optional SQL/PostgreSQL-style source layer documentation

## Known Limitations

- The public repository includes selected report page screenshots rather than a downloadable finished dashboard product.
- The PBIX/PBIP file is not included in this public v0 because layout and navigation are still being refined.
- The processed dataset is anonymized and should be interpreted as portfolio demonstration data, not as a public operational dataset.
